# Adatelemzés 1. Képletgyűjtemény és Puska

Ez a projekt az Adatelemzés 1. tárgyhoz készült képletgyűjteményt és puskát tartalmazza.

## Telepítés és Futtatás

A projekt Docker Compose segítségével futtatható:

```bash
docker compose up -d
```

Az oldal a [https://adatelemzes.prstart.hu](https://adatelemzes.prstart.hu) címen érhető el (éles környezetben).

## Tartalom

- Leíró statisztika
- Indexszámítás
- Standardizálás
- Valószínűségszámítás
- Becsléselmélet
